(,) Selector list

The selector list (,) selects all the matching nodes. A comma-separated list of selectors represents the union of all elements selected by each of the individual selectors in the list.

Syntax

The comma separates one or more selectors in a list.
element, element, element { style properties }

Invalid selector list

A downside to using comma-separated lists of selectors is that if any selector is invalid, the entire rule is ignored.
