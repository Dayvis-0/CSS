(,) Selector list

The CSS selector list (,) selects all the matching nodes. A selector list is a comma-separated list of selectors.

Syntax

The comma separates one or more selectors in a list.
element, element, element { style properties }

When a selector list contains an invalid selector, the entire style block is ignored, except for the :is() and :where() pseudo-classes that accept forgiving selector lists.
