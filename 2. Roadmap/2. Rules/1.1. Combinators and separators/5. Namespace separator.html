Namespace separator

The namespace separator (|) separates the selector from the namespace, identifying the namespace, or lack thereof, for a type selector.

Syntax

namespace|element { style properties }

Example 1: HTML5 inline SVG (namespaces DON'T work as expected)

In HTML5, elements don't have explicit XML namespaces, so namespace selectors fail.

Example 2: SVG with explicit xmlns (namespaces DO work!)

When SVG has explicit xmlns attribute, namespace selectors work correctly.

✅ This SVG has xmlns="http://www.w3.org/2000/svg" so svg|a works!

Key Takeaways

  • |element — elements with NO namespace
  • *|element — elements in ANY namespace (including none)
  • ns|element — elements in a SPECIFIC namespace
  • In HTML5, inline SVG elements don't automatically get the SVG namespace
  • Add xmlns="http://www.w3.org/2000/svg" to SVG for namespace selectors to work
